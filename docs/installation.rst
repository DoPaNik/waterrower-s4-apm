============
Installation
============


Stable release
--------------

To install waterrower-s4-apm, run this command in your terminal:

.. code-block:: console

    $ pip install waterrower_s4_apm

This is the preferred method to install waterrower-s4-apm, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for waterrower-s4-apm can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/DoPaNik/waterrower_s4_apm

Or download the `tarball`_:

.. code-block:: console

    $ curl -OJL https://github.com/DoPaNik/waterrower_s4_apm/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/DoPaNik/waterrower_s4_apm
.. _tarball: https://github.com/DoPaNik/waterrower_s4_apm/tarball/master
